# История изменений «Проектной основы»

Все заметные изменения самого шаблона документируются в этом файле.

## 0.3.0 — 2026-07-15

### Добавлено

- планируемое и транзакционное обновление существующих проектов из отдельного выпуска;
- исторические SHA-256 управляемых файлов для версий `0.1.0–0.2.0`;
- резервное копирование, отчёт операций и автоматический откат при ошибке проверки;
- явная схема совместимых форматов решений, допущений, вопросов и источников;
- состояние шаблона с исходной, предыдущей и текущей версиями;
- сценарии успешной миграции, конфликтов, отсутствующего маркера версии и отката.

### Безопасность

- канонические материалы проекта не заменяются миграцией;
- изменённые управляемые файлы требуют ручного разрешения через `-ForceManagedFiles`;
- путь каждой операции проверяется на выход из проекта и точки повторного анализа.

## 0.2.0 — 2026-07-15

### Добавлено

- безопасная команда добавления решений, вопросов и источников с автоматическими ID;
- автоматическая ротация устаревших записей `HANDOFF.md` в проектный changelog;
- проверка входящих и исходящих локальных ссылок для заметок в `docs` и `_inbox`;
- генератор игнорируемого дайджеста перед коммитом;
- лёгкий, основной и регулируемый профили ведения проекта;
- сквозные тесты ежедневного цикла, включая параллельное добавление записей.

### Изменено

- ежедневный протокол включён в единую книгу проекта;
- версия распространяемого шаблона повышена до `0.2.0`.

## 0.1.2 — 2026-07-15

### Добавлено

- маркер версии внутри распространяемого шаблона;
- SHA-256 для каждого выпускного ZIP;
- сквозная проверка состава, безопасности, контрольной суммы и развёртывания архива;
- автоматическая сборка артефакта и GitHub Release для тегов `v*`.

### Изменено

- файлы ZIP сортируются, а их внутренние временные метки фиксируются, поэтому одинаковые исходники дают одинаковый SHA-256;
- запись ZIP и SHA-256 выполняется через временные файлы, чтобы не оставлять частичный выпуск;
- версия, changelog, содержимое шаблона и Git-тег проверяются на согласованность.

## 0.1.1 — 2026-07-15

### Исправлено

- исправлена инструкция, ссылавшаяся на несуществующую папку `starter-kit`;
- дата создания единой книги привязана к дате инициализации проекта;
- добавлена календарная проверка даты и безопасное экранирование названия проекта;
- запрещён выход ссылок, источников и выходного файла манифеста за пределы проекта;
- временные каталоги тестов сделаны уникальными для параллельных запусков;
- перезапись выпускного ZIP теперь требует явного параметра `-Force`.

### Добавлено

- лицензия MIT для исходного кода, документации и распространяемого шаблона;
- негативные тесты опасных путей, неверных дат, дублирующихся ID и устаревшей единой книги;
- кроссплатформенная проверка на Ubuntu и Windows;
- единые правила окончаний строк через `.gitattributes`.

## 0.1.0 — 2026-07-15

- опубликовано переносимое ядро проектной базы знаний;
- добавлены инициализация, единая книга и автономные проверки структуры.
