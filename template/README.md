## Быстрый старт

Требуется PowerShell 7 (`pwsh`). Для автоматической обработки PDF, Word, Excel и презентаций дополнительно используется Python 3.10+ и Microsoft MarkItDown.

Файлы `TEMPLATE-VERSION` и `TEMPLATE-STATE.json` фиксируют версию, схему реестров и историю обновления основы.

1. Скопируйте содержимое папки `template` из репозитория «Проектная основа» в новый репозиторий.
2. Выполните `pwsh ./scripts/init-project.ps1 -Title "Название проекта" -Slug "project-slug"`.
3. Пройдите [пошаговый старт](START-HERE.md).
4. Выберите [рабочий профиль](WORK-PROFILES.md) и зафиксируйте выбор решением.
5. Заполните [бриф](PROJECT-BRIEF.md), затем [решения](DECISIONS.md) и [открытые вопросы](OPEN-QUESTIONS.md).
6. Выберите [нейтральный профиль контекста](CONTEXT-WORKFLOW.md) по доступному бюджету модели.
7. При необходимости настройте [обработку вложений](INGESTION-WORKFLOW.md) и [коммерческую нейросеть](AI-CONNECTIONS.md).
8. Настройте состав [единой книги](scripts/project-dossier.manifest.json).
9. Выполните `pwsh ./scripts/build-project-dossier.ps1` и `pwsh ./scripts/validate-vault.ps1`.
10. Зафиксируйте первый контрольный рубеж в [передаче контекста](HANDOFF.md).

## Правила выпуска

1. Канонические файлы изменяются до производных представлений.
2. Выполняется `pwsh ./scripts/validate-vault.ps1`.
3. Выполняются сборка и проверка `PROJECT.md`.
4. Готовится дайджест изменений и запрашивается разрешение владельца на публикацию.

Практические команды добавления записей, ротации истории и подготовки дайджеста собраны в [ежедневном протоколе](DAILY-WORK.md).
Проверяемое обновление существующего проекта описано в [руководстве по миграциям](MIGRATIONS.md).
Локальный пакет контекста без вложений собирается командой `pwsh ./scripts/build-context.ps1 -Profile standard -Check`.
Единый пакет с адресно выбранными фрагментами вложений собирается командой `pwsh ./scripts/build-ai-package.ps1 -Profile standard -ExpandSource S-002 -SourceQuery "тема задачи" -RefreshSources -Check`.
Обе команды работают локально и ничего не отправляют внешним API.
